Python Input and Output

A function is defined as a block of organized, reusable code used to perform a single, related action. Python has many built-in functions. The functions like input() and print() are widely used for standard input and output operations respectively.

taking input using input() function:

In c, scanf() is used to call the input. Same in python, built-in function raw_input() is used to call the function to tell the program to stop (or) wait for user to key in the data.

Example : 
>>> input()
hello!how are you?
Output :
‘hello!how are you’
For example, if you want to assign some variable through input.
>>> num=input()
6
>>> print num
>>> 6

You can do this by writing something in quotes inside the input parentheses. That matter will come in a line and will wait for the user input. You can then type the input and then you press ENTER button,

Example :
>>> num=input(“enter any number”)
enter any number 5
>>> print num
>>> 5

- Here, we can see that the entered value 10 is a string, not a number. To convert this into a number we can use int() or float() functions.

Example :
>>> int(‘5’)
>>> 5
>>> float(‘5’)
>>> 5.0

- The eval() function is used to evaluate even expressions, provided the input is a string.

Example :
>>> eval(‘3*5’)
>>> 15

Output using print() function

If you want display the output then print() function is used. To print arguments write arguments in parenthesis of print() function and press enter key.

Example :
>>> print(“hello”)
>>> hello

You can assign a value to the variable and access that value through variable as following

Example :
>>> a=10
>>> print (‘The value of a is’, a)
>>> The value of a is 10
